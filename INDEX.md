# INDEX.md — Selfdex 导航地图

> **用途**：跨工作区 AI 的「目录 + 关键词索引」。AI 在涉及个人话题的对话中**强制读本文件**，命中关键词后再深读对应主题文件。
>
> **维护规则**：每次新增 / 修改 / 删除 compiled / explored 文件后，**必须同步更新本文件**（详见 `docs/rules/compiled.md` §9）。
