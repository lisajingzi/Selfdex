# 事件档案模板（复制后再填，不要直接改本文件）

> 文件名：`events/<分类>/YYYY-MM-DD-<事件关键词>.md`
> 原则：事实优先，细节优先，不压缩。能写多详细写多详细。

---

## 元信息

- **发生日期**：YYYY-MM-DD（或时间段）
- **分类**：工作 / 感情 / 友情 / 亲情 / 爱好 / 健康 / 心灵 / 财务 / 学习 / 生活 / 其他
- **涉及人物**：[张三](../../people/张三.md)、[李四](../../people/李四.md)
- **地点**：（可选）

## 事件经过

> 还原现场。**有什么写什么，按时间顺序，不要总结。**
> 包括：谁说了什么、做了什么、当时的情境。

-

## 关键细节

> 容易被时间冲淡、但你想留下来的细节（一句原话、一个表情、一个数字）。

-

## 当时的我

> 当时的感受、反应、想法。事实层面，不必"升华"。**🔒 AI 只能复述用户原话，不替用户加感受。**

-

## 后续思考（可选）

> 如果这件事引发了观点或决策，链接到 `mine/raw/` 或 `mine/compiled/` 下的文件。

- 相关思考：[YYYY-MM-DD-主题.md](../../raw/YYYY-MM-DD-主题.md)
